- Cloud & IAM security
- DevSecOps & CI/CD security

All work is built using free tools and focuses on real-world engineering problems:
designing, implementing, breaking, and securing systems.

- Infrastructure as Code (Terraform)
- CI/CD Security (GitHub Actions)
