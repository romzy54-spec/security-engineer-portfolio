# Cloud & IAM Security Labs

This section focuses on identity and access management risks in cloud environments,
with an emphasis on Azure Entra ID (Azure AD).

## Scope
- Role assignments and privilege boundaries
- Identity misconfigurations and abuse paths
- Conditional Access evaluation
- Least privilege design and remediation

## Labs
- Azure IAM role misconfiguration analysis
- Privileged access risk assessment
- Identity hardening recommendations
