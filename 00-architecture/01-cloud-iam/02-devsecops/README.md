# DevSecOps Pipeline Security Labs

This section focuses on embedding security controls into CI/CD pipelines
to detect vulnerabilities early without disrupting developer velocity.

## Scope
- CI/CD security architecture
- Static application security testing (SAST)
- Dependency and secret scanning
- Pipeline risk analysis and remediation

## Labs
- Secure GitHub Actions pipeline with CodeQL
- OWASP Juice Shop CI security analysis
- Pipeline failure and risk triage
